# 短链接生成 
http://url.suitwe.com

## 使用框架
Springboot+SpringDataJpa+Redis+Mysql+LayUi

## 功能说明
提供长链接转短网址功能

1.输入长链，可以生成短链

2.可配置长度、字符集

3.支持自定义短链，可以指定字符串作为短链的key

4.支持访问计数
